# Initial RAG Evaluation Questions

Use these questions to test whether the first knowledge base retrieves useful,
traceable evidence.

1. What factors are associated with recurrence in idiopathic granulomatous mastitis?
2. What baseline clinical features should be collected for non-puerperal mastitis?
3. Which ultrasound findings are commonly reported in granulomatous mastitis?
4. What treatment options are described for idiopathic granulomatous mastitis?
5. How do studies define complete remission, partial response, recurrence, or treatment failure?
6. What evidence exists for corticosteroids, methotrexate, antibiotics, drainage, and surgery?
7. Which variables should be extracted from ultrasound reports for treatment-response prediction?
8. What follow-up time windows are commonly used to evaluate treatment response?
